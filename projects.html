Obstacle Avoiding Robot

This Arduino-powered robot detects obstacles using an ultrasonic sensor and intelligently changes direction. It's an ideal beginner robotics project that teaches the fundamentals of automation, sensors, and motor control.

  • Microcontroller: Arduino Uno
  • Sensor: HC-SR04 Ultrasonic Sensor
  • Motors: 2 DC Motors with L298N motor driver
  • Power: 9V Battery / Power Bank
Home Automation System

Control home appliances with a smartphone via Bluetooth. This system allows users to toggle lights, fans, and more with a single tap using an Arduino and relay module.

  • Microcontroller: Arduino Uno
  • Module: HC-05 Bluetooth
  • Control: Relay Module for AC appliances
  • Interface: Android app (MIT App Inventor)
Arduino Weather Station

This mini weather station displays temperature, humidity, and pressure data on an LCD. Perfect for environmental monitoring, it uses DHT11 and BMP180 sensors with Arduino for accurate results.

  • Sensors: DHT11 (Temp & Humidity), BMP180 (Pressure)
  • Display: 16x2 LCD with I2C
  • Usage: Climate monitoring and data logging
Smart Dustbin

The Smart Dustbin uses an ultrasonic sensor to detect hand proximity and opens the lid automatically using a servo motor. It promotes hygiene and is a great automation project for public spaces and homes.

  • Sensor: HC-SR04 Ultrasonic Sensor
  • Actuator: SG90 Servo Motor
  • Power: 5V via USB or battery
